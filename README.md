# 🚦 Road Accident Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes road accident data using Excel Pivot Tables
to identify trends, accident severity, causes, and high-risk regions.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables & Charts
- Slicers & Timeline

## 📊 Key Insights
- Year-wise accident trend
- State-wise accident distribution
- Major causes of road accidents
- Fatal vs non-fatal accidents

## 📁 Files
- Road_Accident_Analysis.xlsx
- Dataset and dashboard images

- Data Analyst Portfolio
- Road Safety Analysis
- Academic Projects
